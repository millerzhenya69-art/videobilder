# Брендовые ассеты Atlanta VPN

Положи сюда:
- `logo.png` — логотип Atlanta VPN (прозрачный фон, белый текст)
  Бот автоматически подхватит его как overlay вместо текстового бейджа.

- `planet.png` — синяя планета (опционально)
  Если есть — накладывается как overlay в нижней части кадра.

Если файлов нет — бот использует текстовый бейдж и geq-планету (работает без ассетов).
